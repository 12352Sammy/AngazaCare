# AngazaCare

AngazaCare is a mental health monitoring web application built with Flask, SQLite, and Chart.js.

## Setup

Install dependencies:

```bash
pip install flask flask-login flask-sqlalchemy bcrypt
```

Run the app:

```bash
python app.py
```

Visit http://localhost:5000
